# 🤖 Neon Multi-Agent Bot

**Многопоточный ИИ-ассистент с разделением ролей (Main Agent, Visualizer, Calculator).**

Проект демонстрирует профессиональный подход к архитектуре AI-ботов в n8n на основе паттерна "Рой агентов" (Swarm/Multi-Agent). Вместо одного неповоротливого промпта, здесь работают разные специализированные субагенты, каждый из которых отлично делает одну вещь: считает или рисует.

## 📊 Основные возможности
* Главный роутер (Main Agent), который понимает интент пользователя и делегирует задачу нужным специалистам.
* Агент-Калькулятор (Calculator) для точных математических расчётов без галлюцинаций LLM.
* Агент-Визуализатор (Visualizer) для работы с графикой.
* Суб-воркфлоу для инструментов (Tools), обеспечивающих кросс-коммуникацию.

---

## ⚙️ Структура проекта

Как и в лучших корпоративных решениях, код разделен на понятные модули в папке `workflows/`. Каждая роль представлена в двух версиях: на русском (`_ru`) и на английском (`_en`).

### Архитектура:
* 🧠 `main_agent_ru.json` / `_en.json` — Основной мозг, маршрутизатор задач.
* 🧮 `calculator_ru.json` / `_en.json` — Субагент для вычислений.
* 🎨 `visualizer_ru.json` / `_en.json` — Субагент-художник.
* 🛠 `tools_subworkflow.json` — Инструментальный саб-воркфлоу.

---

## 🛠 Технологический стек
* **Оркестрация**: n8n
* **Управление логикой**: Multi-Agent Pattern (Advanced AI Nodes)
* **Языковые модели**: OpenAI GPT-4o / Local LLMs (через Groq/OpenRouter)
