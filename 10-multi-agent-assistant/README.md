# 10-multi-agent-assistant

**Статус:** ✅ Готово  
**Тип:** Multi-Agent System  
**Назначение:** Управление жизнью и бизнесом

## Описание
Система из 5 специализированных AI-агентов для управления задачами, календарём, контактами и почтой.

## Agents
- `Calendar Agent v2` — Google Calendar управление
- `Contact Agent` — Google Sheets контакты
- `Task Agent` — Notion задачи
- `Mail Router` — Gmail интеграция
- `Context Manager` — векторный поиск и анализ

## Стек
n8n, GPT-4, Groq, Supabase, Notion, Google Workspace, Telegram
