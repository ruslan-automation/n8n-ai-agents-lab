# 🤖 Multi-Agent Personal Assistant

**Система многоагентного Email-ассистента на базе n8n.**

Этот проект представляет собой комплексную архитектуру, где входящая почта анализируется и автоматически распределяется между несколькими специализированными ИИ-агентами: Календарем, Менеджером Задач, Контакт-менеджером и другими.

## 📊 Основные возможности
* **Умная маршрутизация**: `mail_router.json` определяет интент входящего письма (встреча, задача, запрос контакта).
* **Календарь-Ассистент**: `calendar_agent.json` ищет свободные слоты времени и создает события.
* **Трекер задач**: `task_agent.json` парсит задачи из писем и переносит их в таск-менеджер.
* **Менеджер контактов**: `contact_agent.json` сохраняет новые контакты.
* **Управление контекстом**: `context_manager.json` сохраняет историю общения до 30 дней.

---

## ⚙️ Структура проекта (Workflows)

Архитектура микросервисная. Каждый агент является отдельным обособленным workflow в `workflows/`, который вызывается главным маршрутизатором как саб-процесс:

* 🔀 `mail_router.json` — Основной мозг, парсит почту и делегирует.
* 📅 `calendar_agent.json` — Обработчик встреч.
* ✅ `task_agent.json` — Обработчик задач.
* 👥 `contact_agent.json` — Обработчик контактов.
* 🧠 `context_manager.json` — Модуль памяти.
* 📧 `email_processor.json` — Обработчик с кнопкой подтверждения.
