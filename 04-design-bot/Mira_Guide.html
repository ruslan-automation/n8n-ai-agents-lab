Telegram

Создание бота и получение токена

⏱️ ~5 мин

Первым делом нам нужно зарегистрировать самого бота в Telegram.

Инструкция:

  1. Откройте Telegram и найдите бота @BotFather.
  2. Нажмите Start и отправьте команду /newbot.
  3. Придумайте имя (например, LUX-Space Assistant) и юзернейм (например, lux_space_bot).
  4. BotFather пришлет вам длинный API Token. Скопируйте его, он понадобится в n8n.
▶️ Смотреть видео-инструкцию (таймкод 03:00)
📥 Куда вставлять в n8n:
  1. В n8n перейдите в меню слева: Credentials.
  2. Нажмите кнопку Add Credential.
  3. В поиске напишите Telegram API и выберите его.
  4. Вставьте скопированный токен в поле Access Token и сохраните.
🧠

AI Мозги

Подключение нейросетей

⏱️ ~10 мин

Для работы "Миры" нам нужны два ключа: один для "мышления" (OpenRouter/GPT-4o) и один для "слуха" (Groq/Whisper).

1. OpenRouter (Мозг) — Как оплатить из РФ 🇷🇺

Мы используем OpenRouter, так как он дает доступ к GPT-4o и Claude 3.5 без VPN. Он принимает крипту, но если вам удобнее платить картой или у вас нет крипто-кошелька, используйте сервис Zarub.

💡
Рекомендуемая модель: google/gemini-2.0-flash-001 — быстрая, умная и с большим бесплатным лимитом. Для старта идеально!
💳 Инструкция по оплате через Zarub:

Этот сервис позволяет выпустить зарубежную виртуальную карту для оплаты OpenRouter, Google и других сервисов.

1. Перейдите в официальный бот Zarub.

2. Пополнение: Минимальная сумма — 10$. Можно пополнить с российской карты или криптой.

Нажмите "Перейти в кошелек" в боте

3. Выпуск карты:

  • В приложении нажмите кнопку "Выпустить карту".
  • Пополните баланс кошелька.
  • Нажмите Зеленую кнопку (перевести на карту), чтобы деньги попали на счет карты.

4. Оплата: Нажмите Фиолетовую кнопку — она покажет реквизиты вашей новой карты. Используйте их при привязке карты в OpenRouter.

Интерфейс: Зеленая кнопка для перевода, Фиолетовая для реквизитов
  • Зайдите на openrouter.ai.
  • Создайте ключ в разделе Keys (Credits -> Add credits для оплаты).
📥 Куда вставлять в n8n:
  1. В n8n зайдите в Credentials → Add Credential.
  2. Найдите OpenRouter API.
  3. Вставьте скопированный ключ (sk-or-...) в поле API Key.
  4. Назовите его OpenRouter API и нажмите Save.

2. Groq (Распознавание голоса) Бесплатно!

Groq делает транскрибацию голосовых сообщений мгновенно. И это полностью бесплатно! Лимиты очень щедрые.

  • Зайдите на console.groq.com.
  • Нажмите Create API Key и скопируйте его.
📥 Куда вставлять в n8n (ВНИМАТЕЛЬНО!):
  1. В n8n создайте Credential типа Header Auth (НЕ Groq API!).
  2. Назовите его: Groq Auth.
  3. В поле Name напишите: Authorization
  4. В поле Value напишите: Bearer ВАШ_КЛЮЧ
    (Сначала слово Bearer, затем пробел, затем ключ).

Google

Подключение Календаря

⏱️ ~15 мин

Самый сложный этап. Нам нужно разрешить n8n управлять вашим календарем.

ВАЖНО: ▶️ Смотреть разбор настройки Google API (таймкод 04:30)

В Google Cloud Console:

  1. Создайте новый проект на Google Cloud Console.
  2. В разделе APIs & Services включите Google Calendar API.
  3. Настройте OAuth Consent Screen (выберите External, укажите свой email).
  4. В разделе Credentials создайте OAuth Client ID.
  5. Скопируйте Client ID и Client Secret.
⚠️
Важно! Не забудьте добавить свой email в Test Users в OAuth Consent Screen. Без этого авторизация не сработает!

В n8n (Настройка Credentials):

  1. Создайте новый Credential: Google Calendar OAuth2 API.
  2. Вставьте Client ID и Client Secret.
  3. Скопируйте OAuth Redirect URL из n8n и добавьте его в настройки Google Cloud Console (Authorized redirect URIs).
  4. Нажмите "Sign in with Google" в n8n, чтобы авторизоваться.
⚙️

Настройка Логики (AI Agent)

⏱️ ~5 мин

Теперь настроим "характер" Миры — её поведение, скрипты продаж и информацию о вашей студии.

Где находится промпт:

Откройте узел AI Agent в workflow. В поле System Message находится весь "мозг" бота.

Что можно (и нужно) изменить:
  • Название студии — замените "LUX-Space" на своё
  • Адрес — укажите реальный адрес вашего шоурума
  • Цены — актуальный прайс на услуги
  • Время работы — когда можно записывать клиентов
  • Имя менеджера — Мира, Анна, Елена...
  • Тон общения — формальный или дружелюбный
💡
Совет: Протестируйте разные формулировки! Иногда замена одного слова в промпте кардинально меняет поведение бота.
🚀
5

Тестирование

⏱️ ~5 мин

Финальный этап — проверяем, что всё работает!

  1. В n8n нажмите Execute Workflow (или включите переключатель Active в правом верхнем углу).
  2. Откройте вашего бота в Telegram.
  3. Напишите /start.
  4. Запишите голосовое: "Давайте запишемся на консультацию на завтра в 15:00".
  5. Проверьте свой Google Календарь — там должна появиться запись!
🔴
Бот не отвечает? Проверьте:
  • Workflow активирован (переключатель Active зелёный)
  • Все Credentials настроены правильно
  • Нет ошибок в вкладке Executions

❓ Частые вопросы и проблемы

Бот не реагирует на сообщения ▼

1. Убедитесь, что workflow активирован (переключатель Active должен быть зелёным).

2. Проверьте вкладку Executions — там видны все запуски и ошибки.

3. Откройте Telegram Trigger ноду и убедитесь, что выбран правильный Credential.

Ошибка "Invalid API Key" в OpenRouter ▼

1. Проверьте, что ключ начинается с sk-or-.

2. Убедитесь, что на балансе OpenRouter есть средства (хотя бы $1).

3. Попробуйте создать новый ключ и подставить его заново.

Google авторизация не работает ▼

1. Добавьте себя в Test Users! Это самая частая ошибка.

2. Проверьте, что OAuth Redirect URL из n8n добавлен в Google Cloud Console.

3. Убедитесь, что Google Calendar API включён в вашем проекте.

Голосовые сообщения не распознаются ▼

1. Проверьте Groq Credential — в поле Name должно быть Authorization.

2. В поле Value должно быть: Bearer ваш_ключ (слово Bearer, пробел, ключ).

3. Убедитесь, что используете тип Header Auth, а не Groq API.

Сколько стоит использование бота? ▼

Groq (распознавание голоса): Полностью бесплатно!

OpenRouter: Зависит от модели. Gemini Flash — ~$0.01 за 100 сообщений. GPT-4o — дороже.

n8n Cloud: От $20/месяц. Self-hosted — бесплатно.

В среднем, для небольшой студии расходы составят $5-15/месяц.

Можно ли изменить имя бота с Миры на другое? ▼

Да! Откройте узел AI Agent и в System Message замените "Мира" на любое другое имя.

Также можно изменить стиль общения: сделать бота более формальным или, наоборот, дружелюбным.
